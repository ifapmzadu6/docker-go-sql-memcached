# docker-go-sql-memcached

# Require

- Docker

# Usage
## 1.Prepare
```
chmod u+x run.sh
chmod u+x remove.sh
chmod u+x mysql_setup.sh
chmod u+x mysql_remove.sh
```


## 2.Run MySQL
```
./mysql_setup.sh
```

## 3.Run Go Server
```
./debug.sh
```

Go Server at https://github.com/ifapmzadu6/Hoppin-Server
